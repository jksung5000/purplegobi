개인정보처리방침

Lingrain — 시행일: 2026년 6월 12일

성준경 (이하 "개발자")는 모바일 애플리케이션 Lingrain(이하 "앱") 이용자의 개인정보를 중요하게 생각하며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 방침은 앱이 어떤 정보를 어떻게 처리하는지 설명합니다.

1. 기본 원칙 — 데이터는 기기 안에 저장됩니다

앱은 회원가입이나 계정 없이 동작합니다. 이용자가 만든 학습 노트, 카드, 복습 기록, 설정값은 모두 이용자의 기기 내부 데이터베이스에 저장되며, 개발자의 서버로 전송·보관되지 않습니다.

2. 처리하는 정보

구분항목목적저장 위치
학습 데이터 노트·카드 내용, 복습 기록, 학습 통계 학습 카드 생성 및 간격 반복 복습 제공 기기 내부
이용자가 입력하는 자료 텍스트, 사진, 녹음 파일 AI 기반 학습 카드 생성 (OpenAI API로 전송하여 처리) 기기 내부 (전송분은 아래 3항 참조)
API 키 이용자가 직접 입력한 OpenAI API 키 이용자 본인의 OpenAI 계정으로 AI 기능 호출 기기 보안 저장소(iOS Keychain 등)

앱은 광고 식별자, 위치 정보, 연락처 등을 수집하지 않으며, 별도의 분석(애널리틱스)·광고 SDK를 사용하지 않습니다.

OpenAI, L.L.C. (미국)

5. 보유 기간 및 파기

  • 기기 내 저장 데이터: 이용자가 앱에서 직접 삭제하거나 앱을 삭제하면 즉시 파기됩니다.
  • 외부 서비스 전송분: 각 서비스의 정책에 따르며, 위 3·4항의 기간을 초과하여 보관되지 않습니다.

6. 이용자의 권리

모든 학습 데이터는 이용자 기기에 있으므로, 이용자는 앱 내에서 언제든 자신의 데이터를 열람·수정·삭제할 수 있습니다. 외부 서비스로 전송된 정보에 관한 문의는 아래 연락처 또는 각 서비스의 개인정보처리방침을 통해 할 수 있습니다.

7. 아동의 개인정보

앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.
